# Oren Research Reports

Daily intelligence briefings generated by the Oren Master Research agent.

Reports are saved as `YYYY-MM-DD.md` and cover:
- Claude Code and Anthropic ecosystem updates
- YouTube channel monitoring
- GitHub repo activity and trending
- Stack updates (Supabase, ElevenLabs, TypeScript, React Native)
- AI agent frameworks and tools
- Kadmon project-specific action items

Run `/oren-master-research` to generate a new report.
